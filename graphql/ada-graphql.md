# Ada GraphQL API

Ada is an AI-powered customer service platform. The API covers bot configuration, conversation management, intents, responses, analytics, handoff to live agents, and multi-channel deployment for automated customer support.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Ada REST API
